## 简介

TrophyCraft 是一款 SJMCL 扩展，让你在启动器中收集成就、追踪使用习惯，并分析你的启动器人格。所有数据均存储在本地，无需联网。

提供两个版本：
- **英文版**（org.yoshino.trophycraft）
- **中文版**（org.yoshino.trophycraft.zh）

## 功能特色

### 🏆 23 个成就，5 大分类

**新手入门**：初次启程 / 启动器居民 / 百战老兵 / 连续三日 / 一周伙伴 / 深夜访客 / 晨型人 / 一日三回

**实例管理**：多实例

**扩展收集**：扩展初探 / 扩展收集者 / 探测与哨兵

**里程碑**：十次启程 / 五十次启程 / 五百次启程 / 一周目 / 满月 / 百日记 / 月度全勤 / 冲刺王

**隐藏成就**：夜猫子 / 早起的小鸟 / 重度使用者

### 🧠 启动器人格分析

基于你的使用数据生成 8 种人格类型。

### 🎮 Minecraft 风格弹窗

成就解锁时弹出仿 Minecraft 进度的弹窗，黑底金边，仪式感满满。

## 使用方法

1. 从 Releases 下载对应的 .sjmclx 文件
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 在首页查看成就卡片，或进入扩展设置查看完整成就墙

## 兼容性

- 最低启动器版本: 1.1.3
